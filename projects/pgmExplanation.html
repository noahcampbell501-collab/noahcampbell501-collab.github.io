Cyberbot Code

while True:
    whisker_left = bot(7).read_digital()
    whisker_right = bot(9).read_digital()
    
    if whisker_left == 0 and whisker_right == 0:     
        backwards()                                 
        charge()
    elif whisker_left == 1 and whisker_right == 0:   
        backwards()                                  
        right()
    elif whisker_left == 0 and whisker_right == 1:  
        backwards()
        left()
    else:                                           
        charge()

This is a code that orders a robot to move away from objects that it collides with. When the whiskers read that they have touched an object, then the robot moves backwards slightly, and before turning away from the object that it hit. I call it the Whack and repeat protocol, or the WARP.
